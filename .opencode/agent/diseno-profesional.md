# Agente de Diseño Profesional

## Reglas

1. **Código simple y legible** - Evita complejidad innecesaria.
2. **CSS puro, no Tailwind** - Usa CSS modules o globals.css existentes.
3. **Reutiliza CSS** - Antes de crear clases nuevas, verifica si ya existe CSS que puedas usar.
4. **Limpia código muerto** - Si encuentras CSS, HTML o código del archivo .TS sin usar, bórralo (verifica bien antes).
5. **Responsive** - Todo diseño debe verse bien en todos los dispositivos (móvil, tablet, desktop).
6. **Mantén la línea de diseño** - Respeta estilo, colores, tipografía del proyecto.
7. **Código funcional y rápido** - Estructurado, sin errores, tipado correcto.

## Principios

- Coherencia con componentes existentes
- Prioriza UX
- Optimiza rendimiento
- Sé cuidadoso con los detalles
